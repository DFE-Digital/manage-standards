Formatting

Headings and subheadings

Use 2 hash symbols followed by a space for a heading, for example:

## This is a heading

To add a subheading, use 3 hash symbols:

### This is a subheading

Try and avoid going beyond 3 levels of headings.

Bullet points

Use bullet points to help words or phrases stand out in your standard.

Copy this example to add bullet points:

Introduce bullet points with a lead-in line ending in a colon:

* leave one empty line space after the lead-in line
* use an asterisk or a dash followed by a space to add an item
* start each item with a lowercase letter, do not end with a full stop
* leave one empty line space after the last item

You can not have sub bullet points in a list as this makes content harder to read.

Numbered steps

Copy this example to add numbered steps: 

1. Leave one empty line space before starting your list.
2. Enter a number followed by a full stop and a space to add an item.
3. Start each item with a capital letter and end it with a full stop.
4. Leave one empty line space after the last item.

You can not have sub number lists as this makes content harder to read.

Inset text

To add inset text (opens in new tab), use a greater than > followed by a space. For example: 

> You must tell us if your circumstances change.

If you need multiple lines of content in an inset text, use the greater than symbol on each line, even for empty lines:

> You must tell us if your circumstances change.
> 
> You can do this by calling us on 0800 123 4567.
> You can also email us at [info@example.com](mailto:info@example.com)

Leave a clear line before and after the inset text to make it stand out.

Links and URLs

To convert text into a link, use square brackets around the link text and round brackets around the full URL. Make sure there are no spaces between the brackets or the link will not work. For example: 

[Additional guidance for standards](https://www.gov.uk/example)
